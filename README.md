

# Small Tool

这是一个基于 Gitee 的项目仓库。

## 项目简介

本项目提供了一个预编译的 Debian 软件包 (`small-tool_0.1.0_all.deb`)，适用于 Debian/Ubuntu 等 Linux 发行版。


## 软件包说明

- **软件包名称**: small-tool
- **版本**: 0.1.0
- **架构**: all (适用于所有架构)
- **格式**: .deb (Debian 软件包)

## 安装方法

使用 dpkg 命令

wget https://gitee.com/veyne0/small-tool/releases/download/0.2.1/small-tool-v0.2.1.deb
sudo dpkg -i small-tool_0.1.0_all.deb
sudo apt install -f
```

## 系统要求

- Debian 系统 (9.x/10.x/11.x 等)
- Ubuntu 系统 (16.04/18.04/20.04/22.04 等)
- 其他基于 Debian 的 Linux 发行版
