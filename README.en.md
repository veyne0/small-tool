# Small Tool

This is an open-source project repository hosted on Gitee.

## Project Overview

This project provides a pre-compiled Debian package (`small-tool_0.1.0_all.deb`) compatible with Debian/Ubuntu and other Linux distributions.

## Package Information

- **Package Name**: small-tool
- **Version**: 0.1.0
- **Architecture**: all (compatible with all architectures)
- **Format**: .deb (Debian package)

## Installation Methods

### Method 1: Using the dpkg command
```bash
sudo dpkg -i small-tool_0.1.0_all.deb
```

### Method 2: Using the gdebi tool (automatically resolves dependencies)
```bash
sudo gdebi small-tool_0.1.0_all.deb
```

### Method 3: Using the apt command
```bash
sudo apt install ./small-tool_0.1.0_all.deb
```

## System Requirements

- Debian systems (9.x, 10.x, 11.x, etc.)
- Ubuntu systems (16.04, 18.04, 20.04, 22.04, etc.)
- Other Debian-based Linux distributions
